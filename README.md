Creé una aplicación “Calculadora Básica” utilizando Windows Forms en C#. La aplicación nos deja introducir dos números y realizar operaciones matemáticas básicas como suma, resta, multiplicación y división.
La interfaz gráfica fue creada utilizando controles como TextBox, Label y Button.
